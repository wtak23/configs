7.1.2.1. references¶

  • Linux Mint 17 is based on Ubuntu 14.04 (source)
  • lsb_release -a

Download cuda .deb here (Get x86_64 Ubun14.04 deb local)

Then follow the 3 step procedure

  1. sudo dpkg -i cuda-repo-ubuntu1404-7-5-local_7.5-18_amd64.deb
  2. sudo apt-get update
  3. sudo apt-get install cuda
